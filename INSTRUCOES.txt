COMO COMPILAR O APP SEM INSTALAR NADA NO PC
=============================================

Metodo: GitHub Actions (compila online nos servidores do GitHub)

PASSO 1 - CRIAR REPOSITORIO NO GITHUB
--------------------------------------
- Va em https://github.com/new
- Crie um repositorio com qualquer nome (ex: TsPlayer)
- Deixe como PUBLICO
- NAO marque nenhuma opcao (README, .gitignore, etc)
- Clique em "Create repository"

PASSO 2 - ENVIAR O PROJETO
---------------------------
Abra o terminal (PowerShell) na pasta TsPlayer-Android e cole:

git init
git add .
git commit -m "primeira versao"
git remote add origin https://github.com/SEU_USUARIO/SEU_REPO.git
git push -u origin main

Troque SEU_USUARIO pelo seu nome de usuario do GitHub
Troque SEU_REPO pelo nome que voce escolheu

PASSO 3 - ACOMPANHAR A COMPILACAO
-----------------------------------
- Va no site do GitHub (seu repositorio)
- Clique na aba "Actions"
- Voce vai ver um workflow rodando (circulo amarelo)
- Espere ficar verde (cerca de 5-10 minutos)

Se der erro (vermelho):
- Clique no workflow que falhou
- Clique no job "build"
- Role a tela procurando a mensagem de ERRO em vermelho
- Mande uma foto ou copie o texto vermelho para eu corrigir

PASSO 4 - BAIXAR O APK
-----------------------
Quando ficar verde:
- Clique no workflow concluido
- Role para baixo, na secao "Artifacts"
- Clique em "TsPlayer-APK" para baixar o APK

PRONTO! O APK esta compilado e pronto para instalar no celular.
